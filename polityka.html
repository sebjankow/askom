Polityka Prywatności

1. Informacje ogólne

Niniejsza Polityka Prywatności określa zasady korzystania ze strony internetowej hostowanej w serwisie GitHub (dalej: „Strona”). Szanuję prywatność użytkowników i dokładam wszelkich starań, aby była ona chroniona.

2. Administrator danych

Administratorem Strony jest jej właściciel / autor repozytorium GitHub: ASKOM Sebastian Jankowski.

3. Zakres zbieranych danych

Strona nie zbiera, nie przetwarza ani nie przechowuje żadnych danych osobowych użytkowników. W szczególności:

  • Strona nie posiada formularzy kontaktowych,
  • Strona nie umożliwia zakładania kont użytkowników,
  • Strona nie wykorzystuje plików cookies,
  • Strona nie korzysta z narzędzi analitycznych (np. Google Analytics),
  • Strona nie monitoruje aktywności użytkowników.

4. Dane techniczne

Hosting Strony odbywa się za pośrednictwem serwisu GitHub. GitHub może przetwarzać dane techniczne (np. adres IP) zgodnie z własną polityką prywatności. Administrator Strony nie ma dostępu do tych danych ani nie ma na nie wpływu.

Polityka prywatności GitHub dostępna jest pod adresem: GitHub Privacy Statement.

5. Prawa użytkowników

Ponieważ Strona nie zbiera danych osobowych, użytkownikom nie przysługują żadne prawa związane z ich przetwarzaniem w ramach działania Strony.

6. Zmiany polityki prywatności

Polityka Prywatności może zostać zaktualizowana w przypadku zmian w sposobie działania Strony. Aktualna wersja dokumentu zawsze będzie dostępna w repozytorium GitHub.

7. Kontakt

W sprawach związanych z niniejszą Polityką Prywatności można skontaktować się z właścicielem repozytorium za pośrednictwem serwisu GitHub lub bezpośrednio pod adresem e-mail podanym w zakładce Kontakt.
